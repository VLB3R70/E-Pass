GUI
==============

The GUI package implements all the necessary modules to setup a user interface which uses the :doc:`core` package.
This package uses the built-in `Tkinter <https://docs.python.org/3/library/tkinter.html?highlight=tkinter#module-tkinter>`_ module
to create the frames.
